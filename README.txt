OVERVIEW
Bejeweled is a match-3 puzzle game developed in HTML and JavaScript. Inspired by the popular game Candy Crush, Bejeweled challenges players to swap adjacent jewels to form a line of three or more of the same type. The objective is to score as many points as possible within a given time or move limit.

FEATURES
Intuitive Drag and Swap Controls: Easily swap jewels by dragging them with your mouse or finger.
Combo and Cascade Mechanics: Create chain reactions for higher scores.
Power-ups and Special Jewels: Match four or more jewels to create special jewels with unique abilities.
Score and Move Tracking: Keep track of your score and remaining moves.
Responsive Design: Enjoy the game on both desktop and mobile devices.

INSTALLATION
1. Clone the repository:
git clone https://github.com/Andras09/Bejeweled.git
cd bejeweled-game

2. Open the game in your browser:
Simply open the index.html file in your preferred web browser.

USAGE
1. Start the Game:
Open index.html in your web browser.
The game board will be displayed with randomly placed jewels.

2. How to Play:
Swap adjacent jewels by clicking and dragging one jewel onto another.
Form a horizontal or vertical line of three or more identical jewels to clear them from the board.
New jewels will fall from the top to fill empty spaces, potentially creating new matches.

3. Scoring:
Each match increases your score.
Special jewels created by matching four or more jewels score higher points.

4. Winning and Losing:
Achieve the highest score possible before running out of time.
